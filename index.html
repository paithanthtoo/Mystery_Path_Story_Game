Mystery Path

Welcome to Mystery Path

An interactive, browser-based hypertext adventure.

About Mystery Path

This project is an interactive story game developed for the SET08101 Web Technologies module.

It demonstrates DOM manipulation, local storage, and responsive design without external frameworks.

How to Play

  • Read the story text carefully.
  • Select a choice to determine the narrative direction.
  • Your progress is saved automatically.
  • Explore different paths to find all endings!
